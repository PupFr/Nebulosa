Privacy Policy

NEBULOSA BOT - Zoom Meeting Management Integration

Last Updated: July 23, 2025

1. Data Collection

We collect minimal data necessary for Zoom meeting management functionality:

  • Zoom OAuth access tokens (temporary, in-memory only)
  • Meeting information (IDs, titles, times) for management purposes
  • User identification data (Zoom user ID) for API authentication
  • Telegram user ID for bot communication

2. How We Use Your Data

  • Data is used exclusively for meeting automation and management
  • Processing occurs only during active user sessions
  • No data is used for marketing or analytics
  • Data is processed only as requested through Telegram bot commands

3. Data Storage and Retention

  • No persistent data storage - all processing in memory
  • OAuth tokens expire automatically and are not retained
  • Meeting data is accessed only during active sessions
  • No user data is logged or stored permanently

4. Data Sharing

  • No data is shared with third parties
  • Data is never sold or used for commercial purposes
  • Only necessary API calls to Zoom for functionality
  • No tracking or analytics services used

5. User Rights

  • Users can revoke access at any time through Zoom account settings
  • Complete control over authorization and permissions
  • Right to request data deletion (though none is stored)
  • Contact available for privacy concerns

6. Security

  • All data encrypted in transit via HTTPS/TLS
  • OAuth 2.0 standard security protocols
  • No persistent storage reduces data exposure risk
  • Regular security updates and monitoring
